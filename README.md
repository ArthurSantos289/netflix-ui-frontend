# netflix-ui-frontend

Projeto desenvolvido durante a Imersão Front-end da Alura com o objetivo de recriar a experiência visual da Netflix, incluindo seleção de perfis e catálogo personalizado.

🚀 Sobre o projeto

Este projeto consiste na construção de uma interface inspirada na Netflix, permitindo a escolha de perfis de usuário e a exibição de um catálogo adaptado para cada perfil.

Foi desenvolvido com foco em prática de fundamentos do front-end, organização de layout e manipulação de elementos na interface.

🛠️ Tecnologias utilizadas

HTML5

CSS3

JavaScript


⚡ Funcionalidades

Seleção de perfis de usuário

Interface inspirada na Netflix

Catálogo personalizado por perfil

Navegação simples e intuitiva


📚 Aprendizados

Durante o desenvolvimento deste projeto, foi possível:

Reforçar conceitos de estruturação com HTML

Trabalhar estilização e responsividade com CSS

Aplicar lógica e interatividade com JavaScript

Melhorar a organização de código em projetos front-end
